# CrudApp

This project was developed in Angular. It is a webpage that uses the REST API from Firebase to add, update and delete information about heroes.

## Angular items
I am using different items of Angular for example:
+ Components
+ Services
+ Custom routes

## Libraries
+ Bootstrap
+ Font-Awesome
+ Sweet Alert
